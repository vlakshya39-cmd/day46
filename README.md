# 🤖 Day 46 – Autonomous Agent Studio

An AI-powered **Autonomous Agent Studio** that orchestrates multiple specialized agents to collaboratively generate, evaluate, critique, improve, and review Python code through an iterative workflow.

Built as part of the **#60DaysClaudeChallenge** using Claude.



---

## ✨ Features

- Multi-agent orchestration workflow
- Planner agent for algorithm design
- Executor agent for Python code generation
- Evaluator agent with scoring system
- Critic agent for identifying weaknesses
- Improver agent for iterative refinement
- Memory Manager for preserving lessons across rounds
- Safety Monitor for detecting unsafe code
- Final Reviewer for producing the final verdict
- Interactive execution history
- Agent activity log
- Pipeline visualization
- Configurable stopping conditions
- Responsive single-file HTML application

---

## 🧠 Agent Architecture

The application consists of eight autonomous AI agents:

- 📋 Planner
- 💻 Executor
- 📊 Evaluator
- 🔍 Critic
- 🔧 Improver
- 🧠 Memory Manager
- 🛡️ Safety Monitor
- ✅ Final Reviewer

These agents collaborate in an iterative orchestration loop until one of the stopping conditions is satisfied.
